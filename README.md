# spring-boot-angular-blank
Blank application with spring boot and angular

### Build

```
mvn package
```

### Run

```
java -jar app-server-0.0.1-SNAPSHOT.jar
```

### Browse

Application
http://localhost:8080

h2 web ui
http://localhost:8080/h2

swagge web ui
http://localhost:8080/swagger-ui.html
